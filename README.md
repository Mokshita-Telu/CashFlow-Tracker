# 💸 SpendTrack

A responsive **Student Expense & Income Tracker** built using HTML, CSS, Bootstrap, and JavaScript.

This web application helps you manage your personal finances by tracking income and expenses, calculating your balance in real-time, and giving useful insights into your spending habits.

---

## 🚀 Features

* ➕ Add income and expense transactions
* ✏️ Edit and 🗑 delete transactions
* 💰 Real-time balance calculation
* 📊 Monthly income and expense summary
* 🔍 Filter transactions by type, category, and date
* 🌙 Dark mode support
* 💾 Data stored using localStorage (no backend required)

---

## 🧠 How It Works

* All transactions are stored in your browser using **localStorage**

* Balance is calculated automatically:

  **Balance = Total Income - Total Expenses**

* Filters help you analyze spending patterns over time
